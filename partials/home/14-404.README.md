# 14-404 · 404 page (EN + ES)

Editorial dark error page. Restrained, branded, useful.

## Files
- `/404.html` — English. Served by Cloudflare on unmatched routes.
- `/es/404.html` — Spanish mirror. ES copy, `<html lang="es">`.

## Self-contained on purpose
Each file ships its own `<style>` block (~12 rules) + `/styles/main.css` for
tokens/base/fonts. No new CSS module added.

Why inline:
- Rare-traffic page — a dedicated module would bloat `main.css` everywhere.
- Avoids coordinating with Agent 12 (home-css) over `main.css`.
- Rules still read `_tokens.css` vars, so the dark surface, type scale and
  gold accent stay in lockstep with the brand.

## Layout
Single `<main>`. No nav, no footer. Restraint over redirection. Four onward
links: home, journal, contact anchor, language switch. `noindex` always.
