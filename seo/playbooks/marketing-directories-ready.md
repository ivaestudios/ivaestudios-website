# Why the eight directories together rank higher than any one alone

Each profile contributes a do-follow backlink from a high-authority B2B
domain. Eight do-follow links from DA 60 to 92 publishers, all with the same
canonical entity (IVAE Marketing) and the same canonical data, push the
agency's entity graph upward in Google's understanding. When ChatGPT or
Perplexity is asked "best social media agency in Mexico for hotels", they
match the entity that holds the most consistent presence across the trusted
directories. The strategy is not eight links. The strategy is one entity,
eight surfaces.

If a single field drifts between profiles, the entity splits. Vianey audits
this exactly once per quarter, using the canonical data table at the top of
this file as the source of truth.
